# Staging Data
The staging layer contains datasets generated by the Power Query ETL process and used for SQL Server data loading.

Staging datasets include:
- stg_amazon_sales
- stg_amazon_ads
- stg_meta_ads

Due to repository size considerations, the complete staging datasets are not stored in this repository.

Full staging datasets are available via [Google Drive](https://drive.google.com/drive/folders/1mTBKgXuo3YKGeyc9Ucy4zg40JoxC3vsp?usp=sharing)
