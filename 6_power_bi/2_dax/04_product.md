# 04 Product Measures
This folder contains measures used for product performance analysis, profitability benchmarking, contribution analysis, and product ranking.

---
## Revenue Contribution Analysis
### Color Gross Revenue Contribution %
```DAX
Color Gross Revenue Contribution % =
DIVIDE(
	[Total Gross Revenue],
	CALCULATE(
    	[Total Gross Revenue],
    	ALL(dim_product[color])
	)
)
```
**Purpose:** 
Calculates the percentage contribution of each product color to total revenue.

## Product Benchmarking
### Color Profit vs Avg
```DAX
Color Profit vs Avg =
IF(
	HASONEVALUE(dim_product[color]),
	VAR AvgProfit =
    	AVERAGEX(
        	ALL(dim_product[color]),
        	[Gross Profit]
    	)

	RETURN
    	[Gross Profit] - AvgProfit
)
```
**Purpose:** 
Compares product profitability against the average gross profit across all colors.

## Top Performing Products
### Top Gross Revenue Color
```DAX
Top Gross Revenue Color =
VAR _TopColor =
TOPN(
	1,
	VALUES(dim_product[color]),
	[Total Gross Revenue],
	DESC
)

RETURN
CONCATENATEX(
	_TopColor,
	dim_product[color]
)
```
**Purpose:** 
Returns the color generating the highest revenue.
